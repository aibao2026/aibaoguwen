# 数据安全、数据库加密和云备份

AI保顾问是本地优先工具。客户、保单、提醒和完成状态默认保存在本机 SQLite：

```text
data/customer-reminders.sqlite
```

这个文件可能包含客户姓名、手机号、证件号、保单信息、保费和跟进状态。不要提交、打包或外发 `data/`。

## 本地数据库加密

维护页可以启用数据库加密。启用后：

- 打开已加密数据库前需要输入数据密码。
- 已有明文数据库会先生成一份迁移前备份，再转换为加密库。
- 基于加密数据库生成的本地备份仍需要数据密码才能打开。
- 修改数据密码时必须输入当前数据密码；当前密码错误不会改密。

访问密码和数据库密码不是一回事：

- `CUSTOMER_REMINDERS_PASSWORD` 只保护网页入口。
- 数据库加密使用维护页里设置的数据密码。

## 本地备份

本地备份默认保存在：

```text
data/backups/
```

导入、清空数据和从云端恢复前都会优先生成本地备份。`data/backups/` 可能包含敏感客户数据，不应进入开源仓库或分享包。

## 坚果云云备份

维护页支持额外上传一份加密备份到坚果云 WebDAV。它的定位是“额外云备份”，不是多设备实时同步。

云备份规则：

- 数据库未启用加密时，不允许连接或上传云备份。
- 云端只上传加密备份文件。
- 从云端恢复会覆盖当前本地数据库，恢复前会自动再做一次本地备份。
- 重复备份不会改变本机工作日历的日常使用。

## WebDAV 应用密码

坚果云 WebDAV 不能使用官网登录密码，也不能使用验证码。需要单独生成第三方应用密码。

获取方式：

1. 登录坚果云官网。
2. 进入账户信息，打开安全选项。
3. 找到第三方应用管理或应用密码。
4. 新建应用密码，名称可填 `客户提醒备份`。
5. 复制生成出来的那串密码。
6. 回到 AI保顾问，账号填坚果云邮箱，密码填刚生成的应用密码。

WebDAV 地址保持：

```text
https://dav.jianguoyun.com/dav/
```

## 应用密码保存边界

坚果云应用密码默认只用于本次连接，不写入本地 SQLite 数据库。

如果用户勾选“记住在本机浏览器”，应用密码会保存到当前浏览器 Cookie。清理浏览器数据、换浏览器或换电脑后需要重新输入。

不要把坚果云账号、应用密码、session cookie、完整手机号、完整证件号或真实客户资料写进日志、文档、截图、测试快照或公开 issue。

## 开源发布提醒

开源发布区不应包含：

- `data/`
- `data/backups/`
- `*.sqlite`
- `.env`
- 真实客户 Excel / PDF / 截图
- 坚果云账号或应用密码
- 飞书 Base 标识、token 或 session cookie

公开发布前请运行白名单导出，并在公开发布区重新测试和扫描。详见 `docs/OPEN_SOURCE_RELEASE_PROCESS.md`。
