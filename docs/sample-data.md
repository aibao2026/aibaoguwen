# 导入表字段说明

开源仓库不附带真实客户 Excel。请使用合成数据或自己的本地数据测试导入。

## 客户信息表

工作表名称：

```text
客户信息
```

当前导入器识别的字段：

| 字段 | 说明 | 示例 |
| --- | --- | --- |
| 客户姓名 | 必填 | 张三 |
| 证件号 | 建议填写，用于严格匹配 | 110101199001010010 |
| 出生日期 | 建议填写，用于生日和年龄型续期 | 1990-01-01 |
| 手机号 | 可选，网页和飞书侧默认脱敏展示 | 13800000000 |

## 保单续期/业绩明细表

工作表名称：

```text
结果
```

当前导入器识别的字段：

| 字段 | 说明 | 示例 |
| --- | --- | --- |
| 投保人 | 必填 | 张三 |
| 投保人证件号 | 建议填写，可为脱敏形态 | 1101********0010 |
| 被保人 | 必填 | 张三 |
| 被保人证件号 | 建议填写，可为脱敏形态 | 1101********0010 |
| 保单号码 | 建议填写，用于保单稳定识别 | P20260001 |
| 保险公司 | 可选 | 某某人寿 |
| 保险产品 | 必填 | 年金保险A款 |
| 规模保费 | 可选，优先作为保费 | 12000 |
| 标准保费 | 可选，规模保费缺失时使用 | 12000 |
| 缴费方式 | 可选 | 年缴 |
| 缴费期间 | 必填或进入待确认 | 10年 / 60周岁 |
| 生效时间 | 必填或进入待确认 | 2020-06-18 |

## 匹配规则

- 客户识别优先使用姓名 + 证件号严格匹配。
- 完整证件号和脱敏证件号匹配时，必须姓名相同，且前后位同时匹配。
- 不确定的身份匹配进入待确认，不自动合并。
- 续期只相信明确缴费期间字段，不从产品名或经验猜缴费年限。

## 数据安全

不要把真实客户 Excel、真实数据库、飞书 Base 标识、session cookie 提交到仓库或发给他人。
